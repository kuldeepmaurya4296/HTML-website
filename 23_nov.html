1.Document Title:

The < title> tag within the < head> section sets the title of the HTML document, which is displayed in the browser's title bar or tab.

2.Sections and Headings:

The < body> section contains the main content of the document. Content is often organized into sections using structural elements like < header>, < footer>, < nav>, and < article>. Headings (< h1> to < h6>) define the hierarchy of the document's sections.

BODY TAG ATTRIBUTES
1. background:
Specifies the background image or color for the document. Example: < body background="background.jpg">
2. bgcolor:
Specifies the background color of the document. Example: `< body bgcolor="#ffffff">`
3. text:
Sets the default text color. Example: < body text="#000000">
